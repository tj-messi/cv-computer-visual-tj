#项目阅读方式

---
##requirement
* Python 3.5+
* Pytorch 0.4.0+
* Numpy
* Matplotlib
* Scikit-learn
* Ubuntu 16.04+
会排布出你需要的环境，一般会在一个requirement.txt文件中列出

此时只需要pip install -r requirement.txt即可

注意可能会出现txt文件中名字和github上面实际的名字不符合的情况，记得在txt中改正后再次进行pip install -r requirement.txt
